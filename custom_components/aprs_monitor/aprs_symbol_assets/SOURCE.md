# APRS symbol graphics attribution

The PNG sprite sheets in this directory are the aprs.fi high-resolution APRS
symbol set by Heikki Hannikainen, OH7LZB:

https://github.com/hessu/aprs-symbols

The upstream project explicitly makes the set available to APRS applications
and requests a link to its source. Detailed per-symbol copyright and licensing
information is preserved unchanged in `COPYRIGHT.md` beside these files.
